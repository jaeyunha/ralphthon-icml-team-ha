# Conformance Comparison Phase

## Inputs

Frozen paper specification, frozen official repository, frozen clean-room implementation, and observed sandbox results.

## Work

Compare all four sources for hidden preprocessing, operation order, unstated hyperparameters, defaults, equation/code divergence, undocumented approximations, and paper insufficiency. Material negative findings use official inspection plus independent clean-room/observed confirmation when possible.

## Completion

Every difference is classified or explicitly inconclusive. Findings are schema-valid, anchored, include methods/limitations/confidence/confirmation paths, and contain no ICML score or recommendation fields.
