# Rebuttal Phase Specification

Entry requires frozen official reviews. The phase subscribes to each publication and processes whichever arrives first while allowing separate transient workers to draft threads in parallel. The persistent coordinator owns the complete response matrix, evidence catalog, commitments, admitted limitations, cross-review consistency, and immutable publication. Completion requires one schema-valid rebuttal per official review, every weakness and key question addressed or honestly unresolved, all threads settled, and no truthfulness or consistency violation.
